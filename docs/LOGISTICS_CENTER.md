# 🦝 [小浣熊ERP官网](https://samforo.icu/) - 物流中心 使用教程

## 使用步骤

1. 📥 下载模板: [物流中心导入模板.xlsx](https://github.com/68110923/chrome_plugins/raw/main/docs/image/LOGISTICS_CENTER/物流中心导入模板.xlsx) *(右键另存为下载)*
2. ✏️ 编辑模板表格字段
    - **订单号**: 唯一不能重复,每个订单号只能生成一个物流号 (物流面单显示)
    - **收件人信息**: 收件人姓名,手机号,地址,邮编 等等  (物流面单显示)
    - **预计到货日期**: 格式为 2023-11-09 (物流签收日期, 需要大于当前日期)
3. 🛍️ 购买物流号:
    - `待发货物流` **导入** → 勾选订单 → 点击 **购买JC物流号:无轨迹** (流转至`无轨迹物流`)
4. ➕生成轨迹:
    - `无轨迹物流` 勾选订单 → 点击 **添加轨迹** → 选择需要的**轨迹类型** → **提交** (流转至`已揽收物流`或`已发货物流`)

## 轨迹说明

* **特殊轨迹**: `已揽收`
  - 表示快递员已经揽收, 但是没有后续轨迹
* **正常签收场景**: `正常轨迹`
  - 表示快递员正常揽收, 并且补全了物流轨迹
* **异常签收场景**: `妥投失败`,`收件人拒收`,`收件人联系不上`,`无有效地址`,`货物已退回给发件人`
  - 表示快递员正常揽收, 并且补全了物流轨迹, 但是最后一步本应该签收的轨迹改为异常场景
