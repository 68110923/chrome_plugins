# TikTok 访问指定国家的商品

配置好后,后续访问TikTok商品页面,都可以生效

### 三. 清除浏览器原本的TK的缓存
[访问链接清除缓存](chrome://settings/content/all?searchSubpage=tiktok): chrome://settings/content/all?searchSubpage=tiktok
![](/docs/image/PROXY_TIKTOK/1a53e02c-53c4-47a8-a9c0-71ce8ff8ed18.png)
### 四. 刷新TK页面
访问TikTok页面，看是否生效。如果生效，说明配置成功。

https://www.tiktok.com/t/ZTHKx9GRMg4kg-fFmud/
