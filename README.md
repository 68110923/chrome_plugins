## 🐵 安装篡改猴

首先，您需要安装"篡改猴"（Tampermonkey）扩展程序，这是一个用户脚本管理器，用于管理和运行各种用户脚本。

1. 访问Chrome应用商店中的篡改猴页面：[篡改猴 - Chrome 应用商店](https://chromewebstore.google.com/detail/%E7%AF%A1%E6%94%B9%E7%8C%B4/dhdgffkkebhmkfjojejmpbldmpobfkfo?hl=zh-CN&utm_source=ext_sidebar)
2. 点击"添加到Chrome"按钮进行安装
3. 安装完成后，您会在浏览器右上角看到篡改猴的图标
4. 进入扩展程序页面[chrome://extensions/](chrome://extensions/)
5. 找到"篡改猴"扩展程序，点击"详情"
6. 勾选以下选项 允许运行用户脚本  固定到工具栏   在无痕模式下启用  允许访问文件网址
